## 📋 所需素材清单

### 1. 勇者伐魔模式 (Boss Battle)

#### 勇者角色
- **Idle 动画** (待机)
  - 尺寸: 32x32 或 64x64 像素
  - 帧数: 4-8 帧
  - 格式: GIF (透明背景)
  - 风格: 像素风格勇者/骑士

- **Attack 动画** (攻击)
  - 尺寸: 32x32 或 64x64 像素
  - 帧数: 4-8 帧
  - 格式: GIF (透明背景)
  - 动作: 挥剑/施法

#### Boss 角色
- **Idle 动画** (待机)
  - 尺寸: 48x48 或 64x64 像素
  - 帧数: 4-8 帧
  - 格式: GIF (透明背景)
  - 风格: 怪物/恶龙/魔王

- **Hit 动画** (受击)
  - 尺寸: 48x48 或 64x64 像素
  - 帧数: 2-4 帧
  - 格式: GIF (透明背景)
  - 效果: 闪烁/后退

#### 特效
- **攻击特效** (可选)
  - 剑光、火球、闪电等
  - 尺寸: 16x16 或 32x32 像素
  - 格式: GIF (透明背景)

## 📐 技术规格要求

### 文件格式
- **首选**: GIF (支持透明背景和动画)
- **备选**: PNG 序列帧 (可以用工具转换为 GIF)
- **不推荐**: APNG (WPF 支持有限)

### 尺寸建议
- **小型**: 16x16, 24x24 像素 (适合小挂件)
- **中型**: 32x32, 48x48 像素 (推荐)
- **大型**: 64x64, 96x96 像素 (适合大挂件)

### 背景要求
- **必须**: 透明背景
- **格式**: GIF 或 PNG
- **颜色**: 索引色或 RGBA

### 动画要求
- **帧率**: 8-12 FPS (每秒 8-12 帧)
- **循环**: 无限循环
- **流畅度**: 至少 4 帧

### PNG 序列帧转 GIF

#### 1. **ImageMagick** (命令行)
```bash

```

#### 2. **Online GIF Maker**
- 网址: https://ezgif.com/maker
- 功能: 上传多张图片，生成 GIF

## 🎨 自定义素材方案

如果找不到完全合适的素材，可以考虑：

### 方案 1: 委托创作
- **平台**: Fiverr, Upwork, 站酷
- **价格**: $10-50 (简单角色)
- **周期**: 3-7 天
- **优点**: 完全定制，版权清晰

### 方案 2: 使用 AI 生成
- **工具**: Midjourney, DALL-E (生成参考)
- **后期**: 用 Aseprite 像素化
- **优点**: 快速，独特
- **缺点**: 需要后期处理

### 方案 3: 简化设计
- **方案**: 使用简单几何图形
- **实现**: 用 WPF 原生控件绘制
- **优点**: 无版权问题，可自由修改
- **示例**:
  - 勇者: 蓝色方块 + 剑形状
  - Boss: 红色圆形 + 角
  - 猫咪: 橙色椭圆 + 耳朵

```

├── Images/
│   ├── Hero/

│   ├── Boss/

│   │   └── boss_hit.gif
│   ├── Cat/

│   │   └── cat_sleep.gif
│   └── Effects/

```

## ⚖️ 许可协议说明

### CC0 (推荐)
- 完全免费，可商用
- 无需署名
- 可修改

### CC-BY (常见)
- 免费，可商用
- 需要署名作者
- 可修改

### CC-BY-SA
- 免费，可商用
- 需要署名
- 修改后需相同协议

### 自定义许可
- 仔细阅读许可条款
- 确认是否可商用
- 是否需要署名
