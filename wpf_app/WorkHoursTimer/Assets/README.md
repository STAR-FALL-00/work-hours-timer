## 🛠️ 工具推荐

### PNG 序列帧转 GIF
如果下载的是 PNG 序列帧，需要转换为 GIF：

#### 在线工具（推荐）
**ezgif.com**
- 网址: https://ezgif.com/maker
- 步骤:
  1. 上传所有序列帧 PNG
  2. 设置帧延迟（建议 100ms）
  3. 点击 "Make a GIF"
  4. 下载生成的 GIF

#### 桌面工具
**Aseprite** (付费)
- 网址: https://www.aseprite.org/
- 功能: 专业像素艺术编辑器

**GIMP** (免费)
- 网址: https://www.gimp.org/
- 功能: 通用图像编辑器

## 📐 文件规格

### 推荐尺寸
- **勇者**: 32x32 或 64x64 像素
- **Boss**: 48x48 或 64x64 像素
- **猫咪**: 32x32 或 48x48 像素
- **金币**: 16x16 或 24x24 像素

### 文件格式
- **格式**: GIF（支持透明背景和动画）
- **背景**: 透明
- **帧率**: 8-12 FPS

### 文件命名
- 使用小写字母
- 使用下划线分隔
- 例如: `hero_idle.gif`, `cat_run.gif`
