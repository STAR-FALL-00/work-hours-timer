## 🎯 设计原则

### 1. 简洁明了
- 信息密度适中
- 重点突出
- 易于理解

### 2. 像素风格
- 使用像素图像
- 保持清晰度
- 复古游戏感

### 3. 视觉层次
- 计时器最大最显眼
- 资源信息次要
- 状态信息最小

### 4. 色彩搭配
- 金色: 重要信息（计时器、金币）
- 蓝色: 次要信息（经验、进度）
- 红色: 警告信息（Boss 血条）
- 白色: 普通文字

## 📝 设计注释

### 为什么选择这个尺寸？
- 320x140: 足够显示信息，不占用太多空间
- 横向布局: 适合放在屏幕底部
- 固定尺寸: 保持一致性

### 为什么使用 Emoji？
- 快速实现: 不需要等待素材
- 易于识别: 直观表达含义
- 易于替换: 后期可轻松更换

### 为什么两种模式？
- 多样性: 满足不同用户喜好
- 趣味性: 增加游戏化体验
- 扩展性: 未来可添加更多模式
