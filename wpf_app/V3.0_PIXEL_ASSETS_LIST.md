## 🛠️ 技术实现指南

### 1. GIF 播放库

WPF 原生 `Image` 控件不支持 GIF 动画，必须使用第三方库。

#### 安装 NuGet 包
```bash
Install-Package XamlAnimatedGif
```

或在项目中添加：
```xml
<PackageReference Include="XamlAnimatedGif" Version="2.2.0" />
```

#### XAML 代码示例
```xml
<Window xmlns:gif="http://wpfanimatedgif.codeplex.com">
    <Image gif:AnimationBehavior.SourceUri="pack://application:,,,/Assets/cat_run.gif"
           gif:AnimationBehavior.RepeatBehavior="Forever"
           Width="48" Height="48" />
</Window>
```

---

### 2. 消除锯齿（关键！）

像素画在放大时，如果使用默认渲染会变得模糊。必须设置：

#### XAML 属性
```xml
<Image RenderOptions.BitmapScalingMode="NearestNeighbor"
       gif:AnimationBehavior.SourceUri="pack://application:,,,/Assets/hero_idle.gif" />
```

#### 说明
- `NearestNeighbor` 保证像素点清晰锐利
- 这是像素风格的灵魂
- 不设置会导致图像模糊

### 4. 在 WPF 中引用资源

#### 方法 1：嵌入资源（推荐）
1. 将 GIF 文件添加到项目
2. 设置 Build Action 为 `Resource`
3. 使用 Pack URI 引用：
```xml
pack://application:,,,/Assets/Images/Cat/cat_run.gif
```

#### 方法 2：内容文件
1. 设置 Build Action 为 `Content`
2. 设置 Copy to Output Directory 为 `Copy if newer`
3. 使用相对路径引用：
```xml
Assets/Images/Cat/cat_run.gif
```

---

### 5. 动画状态切换

#### 代码示例
```csharp
// 在 WidgetViewModel 中
private void UpdateAnimation(string state)
{
    string animationPath = state switch
    {
        "idle" => "pack://application:,,,/Assets/Images/Cat/cat_idle.gif",
        "running" => "pack://application:,,,/Assets/Images/Cat/cat_run.gif",
        "sleeping" => "pack://application:,,,/Assets/Images/Cat/cat_sleep.gif",
        _ => "pack://application:,,,/Assets/Images/Cat/cat_idle.gif"
    };
    
    // 更新 Image 的 Source
    AnimationBehavior.SetSourceUri(CatImage, new Uri(animationPath));
}
```

## 📞 技术支持

### 常见问题

**Q: GIF 不显示？**
A: 检查是否安装了 XamlAnimatedGif NuGet 包，并正确引用了命名空间。

**Q: 图像模糊？**
A: 确保设置了 `RenderOptions.BitmapScalingMode="NearestNeighbor"`。

**Q: 动画不循环？**
A: 设置 `gif:AnimationBehavior.RepeatBehavior="Forever"`。

**Q: 找不到资源？**
A: 检查 Build Action 是否设置为 `Resource`，Pack URI 是否正确。
