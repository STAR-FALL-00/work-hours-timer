- 详细数据列表

#### 5. 设置功能 ⚙️
- 工作时长设置

- 主题切换
- 数据导出

### 🆕 v1.1.0 新增功能

#### 1. 💰 经济系统
**核心机制**:
- 工作 1 分钟 = 1 金币
- 连续工作 60 分钟 = 额外 50 金币（连击奖励）
- 完成项目 = 大量金币奖励

**界面展示**:
- 主页右上角显示金币余额
- 实时奖励预览（工作时）
- 连击提示（接近 60 分钟时）

**使用场景**:
- 购买商店商品
- 解锁主题皮肤
- 购买功能道具

---

#### 2. 🎯 项目管理（BOSS 系统）
**核心概念**:
- 项目 = BOSS
- 预估工时 = BOSS 总血量
- 实际工时 = 已扣除血量

**主要功能**:
- 创建项目（设置名称、预估工时、描述）
- 选择当前项目（工作前）
- 工时自动分配到项目
- BOSS 血条实时更新
- 项目完成自动结算奖励

**BOSS 血条**:
- 颜色动态变化（绿→橙→红）
- 显示进度百分比
- 显示已用/预估工时

**项目管理**:
- 双标签页（进行中/已完成）
- 项目列表查看
- 项目归档/重新激活
- 项目详情查看

**完成奖励**:
- 金币：预估工时 × 100
- 经验：预估工时 × 100
- 播放胜利音效
- 可能解锁成就

#### 4. 📊 工作热力图
**功能特点**:
- GitHub 风格日历热力图
- 显示过去一年（365 天）工作记录
- 颜色深浅映射工作时长

**颜色映射**:
- 灰色：无工作记录
- 浅绿：1-2 小时
- 中绿：3-4 小时
- 深绿：5-6 小时
- 最深绿：7+ 小时

**交互功能**:
- 点击任意日期查看详情
- 显示当日工作时长
- 格式化日期显示

**统计信息**:
- 过去一年总工时
- 工作天数统计
- 图例说明

**使用场景**:
- 查看工作习惯
- 发现工作规律
- 设定工作目标
- 长期追踪进度

## 🎮 游戏化系统

### 经济闭环
```
工作 → 金币 → 购买 → 提升体验 → 更有动力
  ↓
经验 → 升级 → 成就 → 成就感
  ↓
项目 → 奖励 → 成就 → 继续
```

### 激励机制

**短期激励**（分钟级）:
- 每分钟获得金币
- 实时奖励预览
- 连击奖励提示

**中期激励**（小时/天级）:
- 项目完成奖励
- 等级提升
- 成就解锁

**长期激励**（周/月级）:
- 热力图可视化
- 累计工时统计
- 高级成就解锁

## 🎯 使用流程

### 典型工作流程

```
1. 启动应用
   ↓
2. 查看金币余额和等级
   ↓
3. 选择当前项目（或创建新项目）
   ↓
4. 点击"开始工作"
   ↓
5. 工作中（实时显示奖励）
   ↓
6. 点击"结束工作"
   ↓
7. 查看奖励对话框
   ↓
8. 查看统计数据（热力图）
   ↓
9. 访问商店（如金币充足）
   ↓
10. 购买商品或继续工作
```

### 项目管理流程

```
1. 创建项目
   - 设置项目名称
   - 估算工时
   - 添加描述
   ↓
2. 选择项目
   - 主页下拉框选择
   ↓
3. 开始工作
   - 工时自动分配
   - BOSS 血条更新
   ↓
4. 完成项目
   - 自动检测完成
   - 结算奖励
   - 播放音效
   ↓
5. 查看已完成项目
   - 项目管理页面
   - 已完成标签
```

## 💡 使用技巧

### 快速获得金币
1. 连续工作 60 分钟（连击奖励）
2. 完成项目（大量奖励）
3. 保持每天工作（积少成多）

### 快速升级
1. 增加每天工作时长
2. 完成项目（额外经验）
3. 使用经验加速卡

### 让热力图更"绿"
1. 保持每天工作
2. 增加每天工作时长
3. 避免长时间中断

### 金币管理
1. 保留一定储备
2. 优先购买实用道具
3. 主题和装饰其次
