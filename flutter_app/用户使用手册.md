# 📖 工时计时器 - 用户使用手册

## 欢迎使用工时计时器！

这是一款功能强大的工作时间记录应用，支持标准模式和游戏化模式，让工作记录变得有趣又高效。

## 🚀 快速开始

### 首次使用

1. **启动应用**
   - 双击桌面图标或从开始菜单启动
   - 应用会以标准模式启动

2. **配置设置**
   - 点击右上角的设置图标 ⚙️
   - 设置标准工作时长（默认 8 小时）
   - 输入月薪（可选，用于计算日薪和时薪）
   - 保存设置

3. **开始记录**
   - 点击"开始工作"按钮
   - 计时器开始运行
   - 需要休息时点击"休息"
   - 完成工作后点击"完成"

## 🎮 双模式介绍

### 标准模式

**特点：**
- 简洁专业的界面
- 清晰的数据展示
- 适合日常办公使用

**主要功能：**
- 工作计时
- 休息记录
- 统计图表
- 成就系统

### 游戏模式（冒险者工会）

**特点：**
- RPG 游戏风格界面
- 羊皮纸和木质装饰
- 等级和经验系统
- 金币和成就

**游戏化元素：**
- 🌱 等级系统：从 Lv.1 到 Lv.50
- ✨ 经验值：每工作 1 小时获得 10 经验
- 💰 金币系统：1 元收入 = 1 金币
- 🏆 成就系统：9 个成就等待解锁

**切换方式：**
1. 进入设置页面
2. 找到"游戏模式"开关
3. 打开开关切换到游戏模式
4. 关闭开关返回标准模式

## 📊 主要功能

### 1. 工作记录

**开始工作：**
- 标准模式：点击"开始工作"
- 游戏模式：点击"⚔️ 接受任务"

**休息功能：**
- 点击"休息"暂停计时
- 休息时间不计入工作时长
- 点击"继续"恢复工作

**完成工作：**
- 点击"完成"结束本次工作
- 自动保存工作记录
- 游戏模式下获得经验和金币

### 2. 统计功能

点击右上角的图表图标 📊 查看统计：

**本周统计：**
- 每日工作时长折线图
- 本周总工时
- 工作天数

**本月统计：**
- 每周工作时长柱状图
- 本月总工时
- 工作天数

**本年统计：**
- 每月工作时长柱状图
- 全年总工时
- 工作天数

### 3. 薪资计算

**智能计算：**
- 基于实际工作日计算（非固定 22 天）
- 自动识别节假日和调休
- 实时显示日薪、时薪、今日收入

**设置月薪：**
1. 进入设置页面
2. 输入月薪金额
3. 保存设置
4. 主页面自动显示薪资信息

**计算公式：**
```
日薪 = 月薪 / 本月工作日数
时薪 = 月薪 / (本月工作日数 × 标准工时)
今日收入 = 已工作时长 × 时薪
```

### 4. 成就系统

点击右上角的奖杯图标 🏆 查看成就：

**新手成就：**
- 🎯 初次任务：完成第一次工作记录
- ⏰ 标准打工人：单日工作满 8 小时
- 🎓 见习毕业：达到 5 级

**进阶成就：**
- ⚒️ 熟练工：达到 10 级
- 🔥 全勤战士：连续工作 7 天
- 💰 小富即安：累计赚取 1000 金币

**高级成就：**
- ⚔️ 百小时勇士：累计工作 100 小时
- 💎 财富自由：累计赚取 10000 金币
- 👑 千小时大师：累计工作 1000 小时

### 5. 设置选项

**标准工作时长：**
- 设置每日标准工作小时数
- 用于计算加班和剩余工时
- 默认 8 小时

**固定上下班时间：**
- 可选功能
- 设置规定的上班和下班时间
- 用于计算"距离下班"时间

**月薪设置：**
- 输入月薪金额
- 自动计算日薪和时薪
- 基于实际工作日计算

**游戏模式：**
- 切换标准模式和游戏模式
- 数据在两种模式间共享
- 随时可以切换

### 数据管理

**自动保存：**
- 所有数据自动保存
- 应用重启后自动恢复
- 无需手动备份

**数据位置：**
```
C:\Users\[用户名]\AppData\Roaming\work_hours_timer\
```

**数据包含：**
- 工作记录
- 应用设置
- 冒险者资料（游戏模式）
- 成就进度

## ❓ 常见问题

### Q1: 如何修改已记录的工作时间？
A: 当前版本不支持修改历史记录，请在记录时确保准确。

### Q2: 数据会丢失吗？
A: 不会。所有数据自动保存到本地数据库，应用重启后自动恢复。

### Q3: 可以导出数据吗？
A: 当前版本暂不支持导出功能，计划在未来版本中添加。

### Q4: 游戏模式和标准模式的数据是分开的吗？
A: 不是。两种模式共享相同的数据，只是显示方式不同。

### Q5: 如何重置所有数据？
A: 删除数据文件夹即可：
```
C:\Users\[用户名]\AppData\Roaming\work_hours_timer\
```

### Q6: 应用占用多少空间？
A: 应用本身约 40MB，数据文件通常小于 10MB。

### Q7: 支持多用户吗？
A: 支持。每个 Windows 用户账户有独立的数据。

### Q8: 可以在多台电脑上使用吗？
A: 可以，但数据不会自动同步。需要手动复制数据文件夹。

## 🔧 故障排除

### 应用无法启动

**可能原因：**
- 缺少必需的系统组件
- 数据文件损坏

**解决方案：**
1. 重新安装应用
2. 检查 Windows 更新
3. 删除数据文件夹后重试

### 数据显示异常

**解决方案：**
1. 重启应用
2. 检查设置是否正确
3. 查看统计页面是否正常

### 界面显示不完整

**解决方案：**
1. 调整窗口大小
2. 检查显示器分辨率
3. 确保分辨率至少为 1280x720
