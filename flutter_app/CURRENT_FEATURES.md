# 🎉 当前版本功能说明

## ✨ 已实现的功能

### 1. 核心工时记录 ⏱️
- **开始工作**：一键开始计时
- **休息管理**：支持暂停休息，自动扣除休息时间
- **结束工作**：保存工作记录到本地数据库
- **实时计时**：精确到秒的计时显示

### 2. 智能薪资计算 💰
- **月薪设置**：在设置中输入月薪
- **智能日薪**：根据当月实际工作日自动计算
  - 自动排除周末
  - 自动排除法定节假日
  - 考虑调休补班日
- **智能时薪**：基于实际工作日和标准工作时长
- **今日收入**：实时显示今日已赚金额

#### 示例（2026年2月）
```
月薪：10,000 元
本月工作日：14 天（春节月）
日薪：714.29 元/天
时薪：89.29 元/小时
工作 4 小时后：357.16 元
```

### 3. 节假日识别 📅
已内置 2026 年中国法定节假日：
- ✅ 元旦（1月1-3日）
- ✅ 春节（1月29日-2月4日）
- ✅ 清明节（4月4-6日）
- ✅ 劳动节（5月1-5日）
- ✅ 端午节（6月25-27日）
- ✅ 中秋节（9月26-28日）
- ✅ 国庆节（10月1-7日）

### 4. 统计图表 📊
#### 本周统计
- 每日工时趋势折线图
- 本周总工时
- 详细数据列表

#### 本月统计
- 每周工时柱状图
- 本月总工时
- 工作天数统计

#### 本年统计
- 每月工时柱状图
- 全年总工时
- 工作天数统计

### 5. 美观的界面设计 🎨
- **现代化卡片**：渐变背景、圆角、阴影
- **图标系统**：直观的图标表示
- **色彩系统**：蓝色（工作）、绿色（正常）、橙色（休息）、红色（警告）
- **响应式布局**：适配不同窗口大小

### 6. 设置功能 ⚙️
- **标准工作时长**：默认 8 小时，可自定义
- **固定上下班时间**：可选，用于计算预计下班时间
- **月薪设置**：用于收入统计
- **数据持久化**：所有设置自动保存

## 📱 使用指南

### 首次使用
1. **启动应用**
2. **点击"开始工作"**开始计时
3. **进入设置**配置月薪和工作时长
4. **查看统计**了解工作情况

### 日常使用
1. **上班时**：点击"开始工作"
2. **休息时**：点击"休息"
3. **下班时**：点击"下班"
4. **查看收入**：主页面自动显示今日收入

### 查看统计
1. 点击右上角 📊 图标
2. 切换"本周/本月/本年"标签
3. 查看图表和详细数据

## 💡 特色功能

### 1. 距离下班倒计时
如果设置了固定上下班时间，会显示"距离下班"倒计时

### 2. 本月工作日提示
在收入统计区域显示"本月X个工作日"，让你了解当月工作日数量

### 3. 加班统计
自动计算并显示加班时长

### 4. 连续工作天数
记录你的连续出勤天数（游戏化功能的一部分）

## 🚀 性能特点

- **快速启动**：本地数据库，无需网络
- **实时更新**：计时器每秒更新
- **数据安全**：所有数据存储在本地
- **低资源占用**：Flutter 原生性能

## 📊 数据存储

所有数据存储在：
```
%USERPROFILE%\AppData\Roaming\work_hours_timer\
```

包括：
- 工作记录
- 设置信息
- 冒险者资料（游戏化数据）
