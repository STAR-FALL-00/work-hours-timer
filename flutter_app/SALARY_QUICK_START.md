# 薪资功能快速开始

## 重要提示

由于修改了数据模型（添加了月薪字段），需要重新生成 Hive 适配器并重启应用。

## 启动步骤

### 1. 停止当前应用
如果应用正在运行，请先关闭。

### 2. 重新启动应用
```bash
cd flutter_app
flutter run -d windows
```

### 3. 设置月薪
1. 打开应用后，点击右上角的设置图标 ⚙️
2. 滚动到"薪资设置"卡片
3. 在"月薪"输入框中输入你的月薪，例如：10000
4. 点击底部的"保存设置"按钮

### 4. 查看收入统计
1. 返回主页面
2. 点击"开始工作"按钮
3. 等待一段时间（例如 30 秒）
4. 点击"下班"按钮
5. 在"今日统计"下方会出现"今日收入"区域，显示：
   - 日薪卡片
   - 时薪卡片
   - 今日已赚卡片

## 示例

假设你的月薪是 10000 元，标准工作时长是 8 小时：

- **日薪**：10000 ÷ 22 = ¥454.55
- **时薪**：10000 ÷ (22 × 8) = ¥56.82
- **工作 4 小时后**：4 × 56.82 = ¥227.28

## 如果没有显示收入统计

请检查：
1. 是否在设置中输入了月薪
2. 是否点击了"保存设置"
3. 是否有今日工作记录
4. 尝试重启应用

## 修改月薪

随时可以修改月薪：
1. 进入设置页面
2. 修改月薪数值
3. 点击"保存设置"
4. 返回主页面，收入统计会立即更新

## 取消显示收入统计

如果不想显示收入统计：
1. 进入设置页面
2. 清空月薪输入框
3. 点击"保存设置"
4. 返回主页面，收入统计区域会消失
