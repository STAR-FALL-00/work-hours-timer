# Flutter 版本归档说明

## 📦 归档信息

**归档日期**: 2026-02-27  
**最终版本**: v2.x (Flutter)  
**归档原因**: 技术栈切换到 WPF/C#/.NET 8  
**状态**: 归档保留，不再主动开发

---

## 🎯 归档原因

v3.0 决定从 Flutter 切换到 WPF，主要原因：

1. **性能优化**: WPF 内存占用降低 60%+
2. **原生体验**: 更好的 Windows 11 特效支持
3. **开发效率**: C# 生态更成熟，团队更熟悉
4. **窗口控制**: Win32 API 集成更简单

详细分析请查看：`../V3.0_FLUTTER_VS_WPF_COMPARISON.md`

## 🔄 数据迁移

### 从 Flutter 迁移到 WPF

Flutter 使用 Hive 数据库，WPF 使用 JSON 文件。

**数据位置**:
```
Flutter: flutter_app/data/hive/
WPF:     wpf_app/AppData/data.json
```

**迁移工具**: 将在 WPF v3.0 中提供自动迁移工具

**迁移步骤**:
1. 导出 Flutter Hive 数据
2. 转换为 JSON 格式
3. 导入到 WPF 应用

## 📦 如何使用 Flutter 版本

### 运行开发版本
```bash
cd flutter_app
flutter run -d windows
```

### 构建发布版本
```bash
cd flutter_app
flutter build windows --release
```

### 安装已打包版本
如果有现成的安装包，直接运行 `.exe` 安装程序。
