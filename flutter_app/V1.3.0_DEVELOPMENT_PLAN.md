## 📝 技术要点

### 新增依赖
```yaml
dependencies:
  # Excel 导出
  excel: ^4.0.3
  
  # PDF 导出
  pdf: ^3.10.7
  printing: ^5.12.0
  
  # 文件保存
  file_saver: ^0.2.9
```

### 数据模型扩展
- `Item` 模型（道具）
- `ItemEffect` 模型（道具效果）
- `ThemeConfig` 模型（主题配置）
- `DecorationDisplay` 模型（装饰品显示）

### 服务扩展
- `ItemService` - 道具服务
- `ExportService` - 导出服务
- `ThemeManagerService` - 主题管理服务
- `DecorationDisplayService` - 装饰品显示服务
