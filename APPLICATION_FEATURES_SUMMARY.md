# 工时计时器 - 应用功能完整总结

## 📱 应用概述

**工时计时器** 是一款 Windows 桌面应用，提供两种工作模式：
- **标准模式** - 简洁高效的工时记录
- **游戏模式** - 趣味化的工作体验

---

## 🎯 双模式设计

### 模式切换
- 应用启动时可选择模式
- 可在设置中随时切换模式
- 两种模式数据互通

---

## 📄 页面结构

应用包含 **5 个主要页面**：

### 1️⃣ 主页（工作记录页）

#### 标准模式主页
**功能按钮：**
- 🟢 **开始工作** - 开始计时
- 🟡 **午休** - 暂停计时（午休时段）
- 🔴 **下班** - 结束工作并保存记录

**显示信息：**
- 当前日期和星期
- 当前工作状态（未开始/工作中/午休中/已下班）
- 工作开始时间
- 午休开始时间（如果有）
- 实时工作时长
- 今日累计工作时长

**特色功能：**
- 实时计时显示
- 自动计算工作时长
- 支持午休暂停
- 一键保存工作记录

#### 游戏模式主页
**额外显示：**
- 👤 冒险者信息
  - 当前等级
  - 经验值进度条
  - 称号显示
- 🎯 每日任务
  - 任务列表
  - 完成进度
  - 任务奖励
- 📊 今日统计
  - 工作时长
  - 获得经验
  - 完成任务数

**游戏化元素：**
- 工作获得经验值
- 完成任务获得奖励
- 等级提升解锁称号
- 动态任务系统

---

### 2️⃣ 统计页面

**数据展示：**

#### 📊 月度统计图表
- 折线图显示每日工作时长
- X 轴：日期（1-31 号）
- Y 轴：工作小时数
- 可视化工作趋势

#### 📈 统计数据
- **本月总工时** - 当月累计工作时长
- **本月平均工时** - 每日平均工作时长
- **本周工时** - 本周累计工作时长
- **今日工时** - 今天的工作时长

#### 📅 工作记录列表
- 显示所有历史工作记录
- 每条记录包含：
  - 日期
  - 开始时间
  - 结束时间
  - 午休时长
  - 总工作时长

**操作功能：**
- 📥 **导出数据** - 导出为 JSON 文件
- 📤 **导入数据** - 从 JSON 文件导入
- 🗑️ **删除记录** - 删除选中的记录

---

### 3️⃣ 成就页面（游戏模式专属）

**成就系统：**

#### 🏆 成就分类

**工作时长类：**
- 🌟 新手上路 - 完成第一天工作
- ⏰ 勤奋工作者 - 累计工作 100 小时
- 💪 工作狂 - 单日工作超过 10 小时
- 🎯 百日坚持 - 累计工作 100 天

**连续打卡类：**
- 🔥 连续打卡 - 连续工作 7 天
- 📅 月度全勤 - 一个月工作满 22 天
- 🌈 季度坚持 - 连续工作 90 天

**时间管理类：**
- ⏰ 早起鸟 - 早上 8 点前开始工作
- 🌙 夜猫子 - 晚上 10 点后还在工作
- ⚡ 效率达人 - 单日工作效率超过 90%

**特殊成就：**
- 🎊 完美一周 - 一周工作满 40 小时
- 🏅 月度冠军 - 月工作时长超过 180 小时
- 👑 传奇大师 - 达到最高等级

**成就显示：**
- 已解锁成就（彩色显示）
- 未解锁成就（灰色显示）
- 成就描述和解锁条件
- 成就解锁时间
- 成就进度条（部分成就）

**统计信息：**
- 总成就数量
- 已解锁数量
- 解锁进度百分比

---

### 4️⃣ 设置页面

**基本设置：**

#### 🎨 界面设置
- **模式切换**
  - 标准模式
  - 游戏模式
  - 实时切换，数据保留

#### ⏰ 工作设置
- **标准工作时长**
  - 默认：8 小时
  - 可自定义设置
  - 用于计算加班时长

- **午休时长**
  - 默认：1 小时
  - 可自定义设置
  - 自动从工作时长中扣除

#### 💰 薪资设置
- **时薪设置**
  - 输入每小时工资
  - 用于自动计算收入

- **加班费率**
  - 设置加班工资倍率
  - 默认：1.5 倍

- **薪资计算**
  - 自动计算日薪资
  - 自动计算月薪资
  - 显示加班费

#### 📊 数据管理
- **导出所有数据**
  - 导出为 JSON 格式
  - 包含所有工作记录
  - 包含设置信息

- **导入数据**
  - 从 JSON 文件导入
  - 自动验证数据格式
  - 合并或覆盖现有数据

- **清空数据**
  - 清空所有工作记录
  - 保留设置信息
  - 需要确认操作

#### 🎮 游戏设置（游戏模式）
- **重置游戏进度**
  - 重置等级和经验
  - 重置成就
  - 保留工作记录

- **每日任务刷新**
  - 自动刷新时间设置
  - 手动刷新任务

## 🎮 游戏化功能详解

### 等级系统

**等级范围：** 1-50 级

**称号系统：**
- Lv 1-5: 实习生
- Lv 6-10: 初级员工
- Lv 11-15: 中级员工
- Lv 16-20: 高级员工
- Lv 21-25: 资深员工
- Lv 26-30: 专家
- Lv 31-35: 大师
- Lv 36-40: 宗师
- Lv 41-45: 传奇
- Lv 46-50: 传奇大师

**经验值获取：**
- 工作 1 小时 = 100 经验值
- 完成每日任务 = 额外经验值
- 解锁成就 = 奖励经验值

### 每日任务系统

**任务类型：**
- 工作时长任务（工作满 X 小时）
- 早起任务（X 点前开始工作）
- 连续工作任务（连续工作 X 天）
- 效率任务（保持高效工作）

**任务特点：**
- 每日自动刷新
- 完成获得经验值奖励
- 任务难度随等级提升
- 最多 3 个每日任务

### 成就系统

**成就总数：** 30+ 个

**成就类别：**
- 工作时长类（10+ 个）
- 连续打卡类（8+ 个）
- 时间管理类（6+ 个）
- 特殊成就类（6+ 个）

**成就奖励：**
- 解锁成就获得经验值
- 成就徽章收集
- 成就进度追踪

- 用户设置

## 📈 数据流程

### 工作记录流程
1. 点击"开始工作" → 记录开始时间
2. 点击"午休"（可选）→ 记录午休时间
3. 点击"下班" → 记录结束时间
4. 自动计算工作时长
5. 保存到本地数据库
6. 更新统计数据
7. 更新游戏进度（游戏模式）

### 数据统计流程
1. 读取所有工作记录
2. 按日期分组统计
3. 计算总时长、平均时长
4. 生成图表数据
5. 显示在统计页面

### 薪资计算流程
1. 读取工作记录
2. 计算总工作时长
3. 区分正常工时和加班工时
4. 应用时薪和加班费率
5. 计算总薪资
